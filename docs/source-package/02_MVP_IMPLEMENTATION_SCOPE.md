# MVP 구현 범위

## MVP 목표

초기 MVP는 다음 질문에 압도적으로 잘 답해야 한다.

> 이 단지를 내 조건으로 살 때, 매매가 외에 얼마가 더 필요하고 어떤 리스크가 있는가?

## 시작 지역

- 서울 광진구 자양동
- LAWD_CD: `11215`
- 초기 단지: 자양동 주요 아파트 10~20개

## MVP 포함 기능

- 자양동 단지 검색
- 국토부 실거래가 기반 단지 요약
- 매수 예상가 입력
- 전용면적 입력
- 보유 주택 수 입력
- 실거주 예정 여부 입력
- 생애최초 여부 입력
- 대출 예정 여부 입력
- 취득세 계산
- 지방교육세 계산
- 농어촌특별세 계산
- 중개보수 상한 안내
- 인지세·등기·법무사 비용 안내
- 규제지역 여부 안내
- 대출 규제 개념 안내
- 매수 리스크 리포트
- 근거 보기
- 외부 매물 링크아웃
- 리포트 공유 mock
- 관심단지 저장
- 뉴스레터 구독 mock
- Loading / Empty / Error state
- Toast
- 전문가 질문 복사

## MVP 제외 기능

- 전국 단지 확장
- 실시간 매물 수집
- 매물 가격 예측
- 양도세 정밀 계산
- 종부세 정밀 계산
- 법인 매수 계산
- 증여·상속 계산
- 임대사업자 계산
- 재건축·재개발 권리 분석
- 자동 법령 반영
- 유료 구독
- 전문가 매칭 자동화
- 결제
- PDF 실제 생성
- 카카오 알림톡

## 구현 우선순위

### 1순위

- 프로젝트 세팅
- 디자인 기반 UI
- 조건 입력
- mock 리포트
- localStorage 관심단지
- 기본 DB 스키마/seed

### 2순위

- 규칙 엔진
- 비용 계산 API
- 리포트 저장
- 실거래가 API 캐싱 구조

### 3순위

- 국토부 API 연동
- 법제처 API 연동
- 워치리스트 DB 저장
- 뉴스레터 DB 저장

### 4순위

- 법령 변경 감지
- 관리자 검토 큐
- 뉴스레터 자동 초안
- 프리미엄 리포트/Fake Door

## MVP에서 반드시 검증해야 할 사용자 행동

- 검색한다
- 조건 입력을 끝까지 한다
- 리포트를 읽는다
- 비용 Breakdown을 펼쳐본다
- 근거 보기를 누른다
- 외부 매물 보기를 누른다
- 관심단지를 저장한다
- 리포트를 공유하려고 한다
- 전문가 질문을 복사한다
- 뉴스레터를 구독한다
